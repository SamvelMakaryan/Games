# Tetris

This is simple implementation of Tetis game.
Version 1.

## Requirements

- G++ compiler
- SFML library

## Usage

- Clone this repository
- Move to Tetris directory
- Create build directory and move there
- Give path to CMakeLists.txt file (cmake ..)
- Build (make)
- Run Tetris executable
